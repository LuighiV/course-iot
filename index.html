Información del curso

Descripción

Este curso es una iniciativa para introducir hacia el Internet de las Cosas (IoT) a cualquiera que esté interesado en explorar el conjunto de tecnologías que lo integran. Aunque es una introducción, aborda una serie de temas que permitirán al participante conocer las herramientas que se requiren para poder desarrollar soluciones dentro del ámbito del IoT.

Estructura del curso

El curso se divide en los siguientes temas.

  1. Introducción al IoT.

  2. Electrónica para dispositivos finales.

  3. Programación de sistemas embebidos.

  4. Protocolos de comunicación para IoT.

  5. Plataformas de software para IoT.

  6. Diseño de una solución de IoT.

Materiales del curso

Para el curso se emplea la plataforma de hardware ESP32 así como el sensor DHT22, LEDs, resistencias, cables, router y plataformas de software para IoT.
